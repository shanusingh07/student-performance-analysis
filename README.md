# 📊 Student Performance Analysis

## 📌 Project Overview

This project analyzes a dataset of student marks using Python.
The goal is to explore student performance and visualize subject-wise statistics using data analysis and visualization techniques.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## 📂 Dataset

The dataset contains marks of 50 students in three subjects:

* Math
* Science
* English

Example:

| Name | Math | Science | English |
| ---- | ---- | ------- | ------- |
| Aman | 78   | 85      | 82      |
| Ravi | 90   | 88      | 91      |

## 📈 Features

* Load dataset using Pandas
* Perform statistical analysis
* Calculate subject-wise average marks
* Visualize results using graphs

## 📊 Visualization

The project generates a bar chart showing the **average marks for each subject**.

## ▶ How to Run the Project

1. Install required libraries

```
pip install pandas numpy matplotlib
```

2. Run the Python script

```
python analysis.py
```

3. The program will:

* Display dataset information
* Calculate average marks
* Show a visualization graph
